## Background & Story

Kai is a mid-level Data Engineer who made the switch from software development. He's in the premium compensation tier: **39.0% earn ₱100k+** (vs 15.5% of DA). He uses Python (78.5%), SQL (88.6%), and cloud platforms (AWS 53.5%, Azure 52.1%) daily.

But he's frustrated by the Git/GitHub adoption gap — nearly half of DEs don't use version control, a professional expectation. He also knows Draw.io architecture diagrams are used by only 35.4% of DEs — documentation skills are underdeveloped.

He wants to advance to Senior DE but needs cloud deployment guidance (AWS/Azure), proper version control practices, and architecture documentation skills. He uses GitHub Copilot (16.5%) and Cursor (12.7%) — AI-assisted coding is mainstream in DE work. Databricks assistant (10.1%) is emerging as a specialized DE AI tool.

## Pain Points (Quantified)

### 1. Git/GitHub Adoption Gap
> "I know Python and SQL, but I don't use GitHub properly. My code is just sitting on my laptop."

- **Evidence:** Only 48.8% of DEs use Git/GitHub — version control is standard professional practice; a portfolio without GitHub is a significant gap
- **Impact:** Portfolio looks messy; can't demonstrate collaborative development skills to employers

### 2. Documentation Skills Underdeveloped
> "I build pipelines but I don't document them. No one knows how my architecture works."

- **Evidence:** Draw.io (35.4%) — architecture diagrams are a practical DE deliverable; proficiency signals ability to document and communicate system design
- **Impact:** Can't demonstrate system design thinking; hard to advance to Senior DE without documentation skills

### 3. Hardware Bottleneck for DE Workloads
> "My laptop can't run Docker or Spark. I need VMs (19.5%) or workstations (12.2%)."

- **Evidence:** DE workloads benefit from higher RAM and compute; 19.5% use VMs, 12.2% use workstations
- **Impact:** Most can't run Docker/Spark locally; creates bottleneck for portfolio project depth

### 4. Cloud Deployment Skills Gap
> "I know Python and SQL, but I don't know how to deploy them to AWS or Azure."

- **Evidence:** AWS (53.5%) and Azure (52.1%) are near-mandatory for DE Professionals; a portfolio without a cloud project is a significant gap
- **Impact:** Needs end-to-end pipeline deployed on cloud platform, version-controlled on GitHub, with documentation
