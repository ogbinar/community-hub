Your 8-Week Journey

Weeks 1–2

SQL Basics + Excel/Power Query

  • Learn SELECT, WHERE, JOIN, GROUP BY
  • Master Excel formulas (VLOOKUP, Pivot Tables)
  • Assignment: Write 5 SQL queries against a sample dataset
🖃 Module 1: SQL Basics → · Browse DEP GitHub Repos →
Weeks 3–4

Power BI Fundamentals + DAX Basics

  • Build your first Power BI dashboard
  • Learn DAX measures (SUM, AVERAGE, CALCULATE)
  • Assignment: Build a dashboard from a sample dataset
🖃 Module 3: Excel & Power BI → · Power BI Desktop is free to download
Weeks 5–6

Python pandas + Data Manipulation

  • Load, filter, transform data with pandas
  • Group, aggregate, merge datasets
  • Assignment: Analyze a real dataset and produce insights
🖃 Module 2: Python Fundamentals → · Browse DEP GitHub Repos →
Weeks 7–8

Cloud Deployment + GitHub Portfolio

  • Deploy your Python scripts to AWS/Azure free tier
  • Set up GitHub portfolio with README templates
  • Assignment: Complete portfolio + share on LinkedIn
