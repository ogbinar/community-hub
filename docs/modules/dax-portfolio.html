Project Overview: Sales Dashboard

What You'll Build

A complete Power BI dashboard that analyzes sales data with DAX measures for time intelligence, running totals, and year-over-year comparisons. This project demonstrates the exact skills employers look for.

Sample Dataset Schema

-- Tables
Sales(SaleID, DateID, CustomerID, ProductID, Amount, Quantity)
Customers(CustomerID, Name, Region, JoinDate)
Products(ProductID, Category, SubCategory, UnitPrice)
Dates(DateID, Date, Year, Month, Day, Quarter)

Dashboard Components

  • 📋 KPI cards: Total Sales, YTD Sales, YoY Growth %
  • 📋 Line chart: Monthly sales trend with moving average
  • 📋 Bar chart: Sales by product category
  • 📋 Map: Sales by region
  • 📋 Table: Top 10 customers by sales

Step-by-Step Project Guide

  1. Import your data into Power BI from a CSV or Excel file.
  2. Create relationships: Dates → Sales, Customers → Sales, Products → Sales (one-to-many).
  3. Mark the Dates table as a date table in Power BI.

Star Schema Checklist

- [ ] One fact table (Sales) with numeric measures
- [ ] Multiple dimension tables (Customers, Products, Dates)
- [ ] Relationships: Dimension → Fact (one-to-many)
- [ ] No circular relationships
- [ ] Date table is marked as a date table
  1. Create basic measures: Total Sales, Total Quantity, Average Order Value.
  2. Add time intelligence: YTD Sales, QTD Sales, MTD Sales.
  3. Calculate comparisons: Prior Year Sales, YoY Growth %.

Core DAX Measures

Total Sales = SUM(Sales[Amount])

Total YTD = TOTALYTD([Total Sales], 'Dates'[Date])

Sales Prior Year = 
CALCULATE(
  [Total Sales],
  SAMEPERIODLASTYEAR('Dates'[Date])
)

YoY Growth % = 
DIVIDE(
  [Total Sales] - [Sales Prior Year],
  [Sales Prior Year]
)
  1. Use TOTALYTD, TOTALQTD, TOTALMTD for period-to-date calculations.
  2. Use PARALLELPERIOD, DATEADD for rolling 12-month and custom periods.
  3. Use DATESINPERIOD for flexible date range filtering.

Time Intelligence Examples

Sales Rolling 12 Months = 
CALCULATE(
  [Total Sales],
  DATESINPERIOD('Dates'[Date], 
    LASTDATE('Dates'[Date]), -12, MONTH)
)

Sales First Half YTD = 
CALCULATE(
  [Total Sales],
  FILTER(ALL('Dates'), 
    'Dates'[Month] <= 6 && 
    YEAR('Dates'[Date]) = YEAR(TODAY()))
)
