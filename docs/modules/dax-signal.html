Essential DAX Functions Reference

Function Purpose Example
SUM() Add all values in a column SUM(Sales[Amount])
AVERAGE() Calculate mean value AVERAGE(Sales[Quantity])
COUNT() Count non-blank values COUNT(Sales[OrderID])
COUNTROWS() Count rows in a table COUNTROWS(Sales)
DISTINCTCOUNT() Count unique values DISTINCTCOUNT(Customers[Name])

CALCULATE is the most important DAX function. It evaluates an expression in a modified filter context.

CALCULATE Examples

Total Sales 2024 = 
CALCULATE(
  [Total Sales],
  YEAR('Dates'[Date]) = 2024
)

Sales Region Manila = 
CALCULATE(
  [Total Sales],
  Customers[Region] = "Metro Manila"
)

% of Total = 
DIVIDE(
  [Total Sales],
  CALCULATE([Total Sales], ALL(Customers))
)
Function Purpose Example
TOTALYTD() Year-to-date total TOTALYTD([Total Sales], 'Dates'[Date])
TOTALQTD() Quarter-to-date total TOTALQTD([Total Sales], 'Dates'[Date])
SAMEPERIODLASTYEAR() Same period last year CALCULATE([Total Sales], SAMEPERIODLASTYEAR('Dates'[Date]))
DATESINPERIOD() Custom date range DATESINPERIOD('Dates'[Date], LASTDATE('Dates'[Date]), -12, MONTH)
