Video Script Template

5-Minute Tutorial Script

[0:00-0:15] Hook: "Did you know 80% of data analysts make this SQL mistake?"

[0:15-0:30] Intro: "Hi, I'm [Name], and today we're covering [topic]. This is a common mistake that even experienced analysts make."

[0:30-3:00] Main Content: Explain the concept with live demo. Show both the wrong way and the right way.

[3:00-4:00] Example: Walk through a real-world example using sample data.

[4:00-4:30] Recap: Summarize the key takeaway in 3 bullet points.

[4:30-5:00] CTA: "If this helped, hit like and subscribe for more data tips. Comment below with what topic you want to see next."

Video SEO Tips

  • Title: Include the main keyword (e.g., "SQL JOIN Tutorial for Beginners")
  • Description: Write 200+ words with keywords naturally included
  • Tags: Use 5–10 relevant tags (SQL, tutorial, data analysis)
  • Thumbnail: Create eye-catching thumbnails using Canva (free)

Content Repurposing Workflow

One Piece of Content → Five Platforms

  1. Create YouTube video: Record a 5-minute tutorial on your topic
  2. Extract key points: List the 3 main takeaways from the video
  3. Write LinkedIn post: Use the template above to create a professional post
  4. Create GitHub README: Document your project with code examples
  5. Post Discord announcement: Share with the community for discussion
  6. Create Twitter thread: Condense key points into a 6-tweet thread
