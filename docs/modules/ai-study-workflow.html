AI Study Workflow Guide

How to use ChatGPT/Gemini/Copilot for debugging, explaining concepts, and reviewing code. Leverage AI as a co-creator in your learning process.

65.8% use AI daily for study AI as co-creator in curriculum

AI Study Workflow: 5 Steps

Use case: You're stuck on a concept (e.g., GROUP BY, DAX CALCULATE). Ask AI to explain it in simple terms with examples.

Prompt Template

Explain [concept] like I'm a beginner. Give me:
1. A simple analogy (non-technical)
2. A real-world example
3. A code example in [SQL/Python/DAX]
4. Common mistakes beginners make

Use case: You need to write a query or script but don't know where to start. Ask AI for a draft, then modify it.

Prompt Template

Write a [SQL query/Python script/DAX measure] that:
- Takes [input data description]
- Performs [specific operation]
- Returns [expected output]
- Handles edge cases like [NULL values/duplicates]

Explain each line of the code after generating it.

Use case: AI-generated code doesn't work. Paste the error message and ask for debugging help.

Prompt Template

I'm getting this error: "[paste error message]"

Here's my code:
[paste your code]

What could be causing this error? Give me 3 possible fixes, ranked by likelihood.

Use case: Your code works but you want to improve it. Ask AI for optimization suggestions.

Prompt Template

Here's my working code: [paste code]

Please review it and suggest:
1. Performance improvements
2. Best practices I'm missing
3. Edge cases I should handle
4. Alternative approaches (if any)

Use case: You've completed a project and need to document it for your portfolio. Ask AI to help write clear documentation.

Prompt Template

Write a README.md for my data project:
- Project description: [brief description]
- Tools used: [SQL/Python/Power BI/etc.]
- What I learned: [key takeaways]
- How to run the code: [setup instructions]

Make it professional but approachable. Include sections for Overview, Setup, Usage, and Results.

Quick-Reference Prompt Templates

Prompt: Explain SQL Concept

Explain [GROUP BY / JOIN / WINDOW FUNCTION] like I'm a beginner. Give me a simple analogy, a real-world example, and a code example in SQL.

Prompt: Debug Python Code

I'm getting this error: "[paste error]". My code is: [paste code]. What's wrong and how do I fix it?

Prompt: Optimize DAX Measure

Here's my DAX measure: [paste measure]. Please review it and suggest performance improvements or alternative approaches.
