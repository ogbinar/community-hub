Micro-Lessons (5 Minutes Each)

Calculated Columns

Calculated columns are computed row-by-row and stored in memory. Use them for values that don't change based on filters.

Example: Calculate Total Price

Total Price = Sales[UnitPrice] * Sales[Quantity]

Measures

Measures are computed on-the-fly based on the filter context. Use them for aggregations that change based on what the user selects.

Example: Calculate Total Sales

Total Sales = SUM(Sales[Amount])

When to Use Each

Use CALCULATED COLUMNS for:
• Values that don't change (e.g., Total Price)
• Filtering/slicing by the value

Use MEASURES for:
• Aggregations (SUM, AVERAGE, COUNT)
• Values that change with filters
• Performance (measures use less memory)

Time Intelligence Overview

DAX provides built-in functions for time-based calculations. These require a properly marked Date table.

Essential Time Intelligence Functions

Total YTD = TOTALYTD([Total Sales], 'Dates'[Date])

Total QTD = TOTALQTD([Total Sales], 'Dates'[Date])

Sales Prior Year = 
CALCULATE(
  [Total Sales],
  SAMEPERIODLASTYEAR('Dates'[Date])
)

YoY Growth % = 
DIVIDE(
  [Total Sales] - [Sales Prior Year],
  [Sales Prior Year]
)

CALCULATE Function

CALCULATE is the most important DAX function. It evaluates an expression in a modified filter context.

CALCULATE Examples

Sales 2024 = 
CALCULATE([Total Sales], 'Dates'[Year] = 2024)

Manila Sales = 
CALCULATE(
  [Total Sales],
  Customers[Region] = "Metro Manila"
)

% of Total = 
DIVIDE(
  [Total Sales],
  CALCULATE([Total Sales], ALL(Customers))
)

Hands-On Exercises

Sample Dataset Schema

-- Tables
Sales(SaleID, DateID, CustomerID, ProductID, Amount, Quantity)
Customers(CustomerID, Name, Region, JoinDate)
Products(ProductID, Category, SubCategory, UnitPrice)
Dates(DateID, Date, Year, Month, Day, Quarter)

Exercise 1: Calculate Total Sales

Create a measure that calculates the total sales amount for all transactions.

Total Sales = SUM(Sales[Amount])

Exercise 2: Calculate YTD Sales

Create a measure that calculates year-to-date sales using the Dates table.

YTD Sales = TOTALYTD([Total Sales], 'Dates'[Date])

Exercise 3: Calculate YoY Growth %

Create a measure that calculates year-over-year growth percentage.

Sales Prior Year = 
CALCULATE([Total Sales], SAMEPERIODLASTYEAR('Dates'[Date]))

YoY Growth % = 
DIVIDE(
  [Total Sales] - [Sales Prior Year],
  [Sales Prior Year]
