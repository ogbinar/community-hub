Project Overview

Analyze sales data using pandas: load CSV, clean data, group by category, calculate metrics, export results to Excel.

Complete Python Script

# Sales Analysis with pandas
import pandas as pd

# Load data
df = pd.read_csv('sales.csv')

# Clean data: remove rows with missing values
df = df.dropna(subset=['Amount', 'Quantity'])

# Group by category and calculate metrics
summary = df.groupby('Category').agg(
    Total_Sales=('Amount', 'sum'),
    Avg_Order=('Amount', 'mean'),
    Num_Orders=('OrderID', 'count')
).reset_index()

# Calculate total
total_sales = df['Amount'].sum()
print(f"Total Sales: ₱{total_sales:,.2f}")

# Export to Excel
summary.to_excel('sales_summary.xlsx', index=False)
print("Exported to sales_summary.xlsx")

Project Overview

Segment customers by spending patterns using pandas: calculate RFM scores, group into segments, create summary tables.

Customer Segmentation Script

# Customer Segmentation with pandas
import pandas as pd

# Load customer data
df = pd.read_csv('customers.csv')

# Calculate RFM metrics
rfm = df.groupby('CustomerID').agg(
    Recency=('OrderDate', lambda x: (pd.Timestamp.now() - x.max()).days),
    Frequency=('OrderID', 'count'),
    Monetary=('Amount', 'sum')
).reset_index()

# Create segments based on percentiles
rfm['R_Score'] = pd.qcut(rfm['Recency'], 5, labels=[5,4,3,2,1])
rfm['F_Score'] = pd.qcut(rfm['Frequency'].rank(method='first'), 5, labels=[1,2,3,4,5])
rfm['M_Score'] = pd.qcut(rfm['Monetary'], 5, labels=[1,2,3,4,5])

# Combine scores into segment
rfm['Segment'] = rfm['R_Score'].astype(str) + rfm['F_Score'].astype(str) + rfm['M_Score'].astype(str)

# Export results
rfm.to_excel('customer_segments.xlsx', index=False)
print(f"Created {len(rfm)} customer segments")

Setup Guide: Running Python on Your Laptop

Step 1: Install Python

  1. Go to python.org/downloads
  2. Click "Download Python 3.x" (latest version)
  3. Run the installer and check "Add Python to PATH"
  4. Click "Install Now" and wait for completion

Step 2: Install pandas & openpyxl

  1. Windows: Open Command Prompt, type: pip install pandas openpyxl
  2. Mac/Linux: Open Terminal, type: pip3 install pandas openpyxl
  3. Wait for installation to complete (usually takes 1–2 minutes)

Step 3: Run Your First Script

  1. Create a new file called hello.py
  2. Add this code: print("Hello, Data Engineering Pilipinas!")
  3. Save the file
  4. Open terminal/command prompt and navigate to the folder containing hello.py
  5. Type: python hello.py (or python3 hello.py on Mac/Linux)
  6. You should see: "Hello, Data Engineering Pilipinas!"

Troubleshooting Tips

• If "python" not found: Add Python to your PATH during installation
• If pip not found: Run "py -m pip install pandas" instead
• If permission error: Try running terminal as Administrator (Windows) or use sudo (Mac/Linux)
• For Jupyter Notebook: Run "pip install jupyter" then "jupyter notebook"
