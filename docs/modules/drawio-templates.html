How to Use Draw.io Templates

  1. Go to Draw.io: Visit app.diagrams.net (free, no account needed).
  2. Create a new diagram: Click "Create New" and choose a blank canvas.
  3. Add shapes: Use the left sidebar to add rectangles, circles, and arrows.
  4. Label components: Double-click any shape to add text labels.
  5. Connect shapes: Click and drag from one shape to another to create connections.
  6. Export as PNG/PDF: File → Export → Save as PNG or PDF.

Draw.io Shape Quick Reference

• Rectangle: Process / Component
• Cylinder: Database / Storage
• Arrow: Data flow direction
• Diamond: Decision point
• Document: Report / Output
• Parallelogram: Input / Output data

Tip: Use the "Cloud" shape from AWS/Azure/GCP libraries for cloud components.
