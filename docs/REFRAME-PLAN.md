# Reframe Plan: Community-Centric Portal

**Date:** 2026-04-23  
**Goal:** Shift from "feature catalog / learning portal" to "community-centric open-source learning hub"  
**Core shift:** Every feature should support **finding**, **building**, and **developing** a community of learning, sharing, and open values (open source software + open learning materials).

---

## Reframing Principles

| Current Framing | New Community-Centric Framing |
|---|---|
| "Data Engineering Pilipinas — Community Hub" | **"DEP Open Commons"** — a community-owned commons of open learning materials, tools, and connections |
| Features as "modules to consume" | Features as **contributable spaces** where anyone can add, improve, share |
| "Learn SQL/Python/Power BI" | "Open-source your learning journey — contribute modules, share projects, build together" |
| Mentorship as "get help" | Mentorship as **"give back"** — structured giving-back framework, reciprocal knowledge sharing |
| Phase roadmap (Months 1–12) | Community **growing seasons** — open invitation to join any phase at any time |
| "Built by Pinoys, for Pinoys" | "Open source learning. Open values. Built together." |

---

## Changes Required

### Change A: Reframe the Main Hub (index.html)

**What changes:**
1. **Title/hero text** — Shift from "structured learning paths" to community-first messaging
2. **Nav structure** — Add "Community", "Contribute", "Open Values" sections
3. **Feature cards** — Reframe each feature as a community-building opportunity (not just consumption)
4. **How It Works** — Reframe from "Watch → Read → Try" to "Join → Contribute → Share → Grow"
5. **Sign-up form** — Reframe from "Join First Cohort" to "Join the Commons / Become a Contributor"

### Change B: Reframe Pathway Page (pathway/index.html)

**What changes:**
1. Title from "Career Shifter DA/DE Pathway" → "Open Learning Pathway — Contribute & Grow Together"
2. Add "How to Contribute" section explaining how anyone can improve modules, add resources, translate content
3. Reframe badges as "community recognition" not individual achievement

### Change C: Reframe Mentorship Page (mentorship/index.html)

**What changes:**
1. Title from "Time-Bound Mentorship Matching" → "Open Knowledge Exchange — Learn & Teach"
2. Add "How to Give Back" section explaining the open-source ethos of knowledge sharing
3. Reframe mentor directory as "Community Contributors" not "mentors/mentees"

### Change D: Reframe Individual Module Pages

**What changes:**
1. Add a "Contribute to This Module" section on every module page with links to GitHub, Discord discussion threads
2. Add "Share Your Work" CTAs encouraging users to submit projects, translations, improvements
3. Reframe AI-interrogated lessons as "Community-vetted content — open source and improvable by anyone"

### Change E: Add New Community Sections Across All Pages

**What changes:**
1. **"Open Values"** section explaining the open-source ethos (free, open, contributable)
2. **"Contribute"** CTAs linking to GitHub repos, Discord channels, contribution guidelines
3. **"Community Voices"** placeholder for user testimonials, project showcases, creator spotlights
