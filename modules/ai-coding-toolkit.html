How to Use GitHub Copilot

  1. Install VS Code: Download from code.visualstudio.com (free)
  2. Install Copilot extension: Search "GitHub Copilot" in VS Code extensions
  3. Sign in: Use your GitHub account to activate Copilot
  4. Start coding: Open a Python or SQL file and start typing

Copilot Prompt Examples

# Type this comment and press Tab to generate code:

# Read CSV file and calculate sales by category
import pandas as pd
df = pd.read_csv('sales.csv')
category_sales = df.groupby('Category').agg(
    Total_Sales=('Amount', 'sum'),
    Avg_Order=('Amount', 'mean')
)
print(category_sales)

Validation Best Practices

  • Always review: Never blindly accept AI-generated code without reviewing it first
  • Test with sample data: Run generated code against a small dataset to verify correctness
  • Check imports: Ensure all required packages are installed before running

How to Use Cursor

  1. Download Cursor: Get it from cursor.com (free tier available)
  2. Create a project: Open a folder with your Python or SQL files
  3. Use Chat: Press Ctrl/Cmd + L to open the AI chat panel
  4. Prompt engineering: Be specific about what you want Cursor to generate

Cursor Prompt Examples

# In the Cursor chat panel:

"Write a Python function that takes a DataFrame and returns 
the top 10 customers by total sales. Include error handling 
for missing columns."

"Explain this DAX measure step by step:
CALCULATE([Total Sales], SAMEPERIODLASTYEAR('Dates'[Date]))"

Prompt Engineering Tips

  • Be specific: Include exact column names, expected output format, and edge cases
  • Provide context: Mention your database type (MySQL, PostgreSQL, etc.)
  • Ask for explanations: Request line-by-line breakdowns of complex queries

How to Use Databricks Assistant

  1. Create a Databricks account: Sign up at databricks.com (free tier available)
  2. Start a notebook: Create a new Python or SQL notebook
  3. Use the assistant: Click the AI icon in the toolbar to open Databricks Assistant
  4. Prompt your task: Describe what you want to accomplish in natural language

Databricks Assistant Prompt Examples

# In Databricks notebook:

"Write a Spark DataFrame transformation that joins two tables 
and calculates year-over-year growth percentage."

"Optimize this PySpark query for better performance on large datasets."

Prompt Engineering Best Practices

Writing Effective Prompts

  • Be specific about the task: Instead of "Write SQL," use "Write a SQL query that calculates monthly sales by product category"
  • Provide context: Mention your database type, table structure, and expected output format
  • Ask for explanations: Request line-by-line breakdowns of complex queries or code
  • Iterate: If the first result isn't perfect, refine your prompt with more details

Prompt Template

Task: [What you want the AI to do]
Context: [Database type, table structure, data sample]
Output format: [Expected result format]
Edge cases: [How to handle NULLs, duplicates, etc.]
Validation: [How to verify the output is correct]

Common Mistakes to Avoid

  • Vague prompts: "Write code" → Too vague, specify the exact task
  • No context: AI doesn't know your table structure or column names
  • Blind trust: Always validate AI output before using it in production
